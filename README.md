# Headless CMS for the Hering App

done with strapi

## Available Scripts

to start it locally:

`yarn develop`


deploy to heroku:

`git push heroku main`

## Endpoints

single

- calendar-page
- start-page

multi

- links
- sections
- chapters
- task
