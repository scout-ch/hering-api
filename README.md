# Headless CMS for the Hering App

done with strapi: https://strapi.io/

## Available Scripts

to start it locally:

`yarn develop`


deploy to heroku:

`git push heroku main`

## Endpoints

single

- calendar-page
- start-page

multi

- links
- sections
  => auto generate slugs
- chapters
  => auto generate slugs
- task

## heroku

available on heroku: https://hering-api.herokuapp.com/admin/
